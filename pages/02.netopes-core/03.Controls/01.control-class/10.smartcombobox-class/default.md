##SmartComboBox class
Class used for rendering Select2 HTML selector.


###Sample code
<pre>
$var = new SmartComboBox([]);
echo $var->Show();
</pre>


###Options list

- ''


###Important remarks

- Do not use as AJAX parameter the 'type' key (internally used to establish return data format: JSON/string/...)
